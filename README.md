# ごきらく亭 公式ホームページ（架空注文サイトのHP化）

実際の店舗（お好み焼き・もんじゃ焼き店「ごきらく亭」）のオーナーからヒアリングを行い、実用性を重視して制作した公式ホームページです。

## 🔗 サイトURL
https://drew-works.github.io/okonomiyaki-order/HP/

## ✨ こだわったポイント・実装のプロセス

1. **実店舗の課題を解決する情報設計と即座の要望対応**
   - オーナーとコミュニケーションを取り、店舗で実際によく聞かれる質問を「よくあるご質問（Q&A）」として実装。
   - 週末の混雑を考慮し、電話予約へスムーズに誘導するための専用セクションとアクションボタンを設置。
   - 公開後にオーナーから要望のあった「高級紅花油の使用」についてのアピールテキストも即座に反映させました。

2. **徹底したレスポンシブ対応（スマホ最適化）**
   - PCだけでなく、店舗を探すユーザーの多くが利用するスマートフォンでの閲覧を想定し、CSSメディアクエリを使用。
   - キャッチコピーがスマホの画面幅で不自然に分裂する問題を解決するため、`white-space: nowrap` と特定のクラスを用いた改行コントロール（`<br class="sp-only">`）を独自に実装し、インパクトのある美しい表示を実現。

3. **Git / GitHub を用いた本番環境へのデプロイ**
   - GitHub Pagesを活用して、実際に一般公開・運用できる環境を構築。
   - Git push時のエラーやバッファサイズの問題も自力でトラブルシューティングし、解決しました。

## 🛠️ 使用技術
- HTML5
- CSS3（Flexbox, Grid Layout, メディアクエリによるレスポンシブアニメーション）
